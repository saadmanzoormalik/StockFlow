# App Store / Play Store Roadmap

## Recommended Path

Start as a mobile-first PWA, then package native shells after core flows work.

## Phase 1: PWA

- Responsive mobile UI
- PWA manifest
- Installable home-screen experience
- Auth, subscription, and portfolio connection working on web

## Phase 2: Native Wrapper

Use Capacitor when the web app is stable:

- `capacitor.config.ts`
- iOS shell
- Android shell
- native splash screen
- app icons
- secure storage review
- deep links

## Phase 3: Store Submission

Requires:

- Apple Developer account
- Google Play Console account
- privacy policy
- terms of service
- financial/educational disclaimer
- subscription disclosure
- account deletion flow
- support URL

## Important Compliance Note

The app must avoid guaranteed-return claims. Use educational framing, expected ranges, risk labels, and clear disclaimers.
